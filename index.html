Font Size and Inline Styling

  • Text Absolute value: 10 px, can be used when the text has to be a certain size
  • TextAbsolute value: 1 cm, can be used when the text has to be a certain size
  • TextRelative value: 1.3rem, which is relative to the root html 16px * 1.3
  • TextRelative value: 0.52 em, relative to parent element(li font-size 40px), 40px*0.52, should be same with the former one
  • TextRelative value: 130%, relative to parent element(li font-size 16px), 16px*1.3, should be same with the former one
  • TextRelative value: 3vw, 3% of the viewport's width
